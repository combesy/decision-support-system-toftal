# Dashboard Gallery

This folder contains screenshots of the interactive Power BI dashboards developed for the TOFTAL Technologies Business Intelligence project.

These dashboards transform operational data into meaningful business insights for managers and decision-makers.

---

# Executive Dashboard

The Executive Dashboard provides a high-level overview of the company's overall performance.

Main KPIs:

- Total Revenue
- Total Orders
- Total Customers
- Total Products
- Revenue Trend
- Sales Distribution

![Executive Dashboard](executive-dashboard.png)

---

# Sales Dashboard

The Sales Dashboard focuses on sales performance and revenue analysis.

It includes:

- Monthly Sales Trend
- Revenue by Category
- Revenue by Country
- Top Selling Products
- Order Analysis

---

# Customer Dashboard

The Customer Dashboard analyzes customer purchasing behavior.

It includes:

- Customer Segmentation
- Top Customers
- Purchase Frequency
- Customer Distribution
- Revenue by Customer

---

# Product Dashboard

The Product Dashboard measures product performance.

It includes:

- Best Selling Products
- Product Categories
- Product Revenue
- Inventory Overview
- Product KPIs

---

# Vendor Dashboard

The Vendor Dashboard analyzes supplier performance.

It includes:

- Vendor Performance
- Purchase Orders
- Supplier Comparison
- Procurement Statistics
- Vendor Contribution

---

All dashboards were developed using **Microsoft Power BI** and connected to the SQL Server Data Warehouse designed for this project.
